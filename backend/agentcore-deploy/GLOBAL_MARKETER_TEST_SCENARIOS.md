# 🌍 Global Marketer 测试场景手册

> 公司背景：**星辰科技 (StarTech)** — 深圳智能家居DTC品牌，核心产品为$49智能家居中控器（支持Matter/Alexa/Google Home/HomeKit），计划2026年Q2进入美国、日本和东南亚市场。

---

## 🌐 类别1：Browser工具 — 网页抓取与竞品分析

### B1: Product Hunt 竞品发现

**提示词：**
```
我们公司星辰科技准备在Product Hunt上发布一款智能家居控制器。请帮我抓取Product Hunt首页，看看今天有哪些热门产品，分析它们的标语和定位策略，给我们的产品发布提供参考建议。
```

**预期工具：** Browser  
**超时时间：** 180秒  
**验证关键词：** Product Hunt, 产品

---

## 🐍 类别2：Code Interpreter — 数据分析与API

**提示词：**
```

- Facebook Ads: 花费$5000, 点击12000, 转化180, 收入$12000
- Google Ads: 花费$3000, 点击8000, 转化95, 收入$7500
- TikTok Ads: 花费$2000, 点击15000, 转化120, 收入$6000
请计算每个渠道的CTR、CVR、CPA、ROAS，并用表格展示。
```

**预期工具：** Code Interpreter  
**超时时间：** 120秒  
**验证关键词：** ROI, ROAS

## 🚀 类别5：端到端业务流程（复杂多步骤）

### E1: 完整出海营销方案

**提示词：**
```
星辰科技是一家深圳智能家居公司，核心产品是$49的智能中控器(支持Matter/Alexa/Google Home)。我们2026年Q2要进入美国市场，总营销预算$100,000。

请帮我制定一个完整的出海营销方案，包括：
1. 市场分析：美国智能家居市场现状和竞争格局
2. 目标用户：用户画像和购买决策路径
3. 品牌定位：差异化定位和品牌故事
4. 渠道策略：线上(Amazon/DTC官网)和社交媒体矩阵
5. 内容策略：各平台内容规划
6. 广告策略：Facebook/Google/TikTok投放计划
7. PR策略：媒体关系和Product Hunt发布
8. 预算分配：$100K的详细分配方案
9. 时间线：Q2三个月的执行节奏
10. KPI：关键指标和目标值
```

**预期工具：** 无  
**超时时间：** 90秒  
**验证关键词：** 市场, 用户, 品牌, 渠道, 预算
