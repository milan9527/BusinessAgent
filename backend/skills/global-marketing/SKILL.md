# 出海营销

为出海团队提供海外市场营销策略制定、多语言内容创作、社交媒体运营和广告投放优化的完整工作流。

## 核心平台策略

### 小红书 (RedNote)
- 定位: 种草平台，适合品牌故事和产品体验分享
- 内容形式: 图文笔记、短视频、直播
- 文案风格: 真实感、生活化、emoji丰富、口语化
- 标签策略: 3-5个热门标签 + 2-3个长尾标签
- 发布时间: 工作日12:00-13:00, 晚间20:00-22:00
- 关键指标: 笔记曝光量、互动率(点赞+收藏+评论)、粉丝增长

### TikTok
- 定位: 短视频娱乐平台，适合病毒式传播
- 内容形式: 15-60秒短视频、直播、TikTok Shop
- 创作要点: 前3秒抓眼球、节奏快、音乐/特效、原生感
- 标签策略: 1-2个挑战标签 + 3-5个相关标签
- 发布频率: 每日1-3条，保持活跃度
- 关键指标: 播放量、完播率、互动率、粉丝增长、GMV(电商)

### Facebook / Instagram
- 定位: 社交+电商，适合品牌建设和精准广告
- 内容形式: 图文帖、Reels短视频、Stories、直播
- 广告体系: 漏斗式投放 (认知→考虑→转化)
- 受众定向: 兴趣定向、Lookalike受众、再营销
- 关键指标: CPM、CPC、CTR、ROAS、频次

### YouTube
- 定位: 长视频平台，适合深度内容和品牌信任建设
- 内容形式: 长视频、Shorts、直播
- SEO要点: 标题含关键词、描述详细、标签精准、缩略图吸引
- 关键指标: 观看时长、订阅转化、CTR(缩略图)

### Google Ads
- 搜索广告: 关键词竞价，适合高意向用户获取
- 展示广告: GDN网络，适合品牌曝光和再营销
- 应用广告 (UAC): 自动化应用推广
- 关键指标: CPC、CPA、ROAS、质量得分

## 广告投放优化

### 核心指标

```
CPI (Cost Per Install) = 广告花费 / 安装数
CPA (Cost Per Action) = 广告花费 / 目标行为数
ROAS (Return On Ad Spend) = 广告收入 / 广告花费
LTV/CAC 比值: 健康值 > 3
ROI = (收入 - 成本) / 成本 × 100%
```

### 素材优化策略

素材类型优先级：
1. UGC风格视频 (真实感强，CTR高)
2. 产品演示视频 (展示核心玩法/功能)
3. KOL合作内容 (信任背书)
4. 图文轮播 (信息量大，适合电商)

素材测试框架：
- 每周产出5-10组新素材
- A/B测试不同hook (前3秒)
- 测试不同CTA文案
- 监控素材衰减周期 (通常7-14天)
- 及时替换疲劳素材

### 投放策略

阶段式投放：
1. 测试期 (1-2周): 小预算多素材测试，找到优质素材和受众
2. 放量期 (2-4周): 优质素材加大预算，扩展相似受众
3. 稳定期 (持续): 控制成本，持续优化，定期更新素材

预算分配建议：
- 测试预算: 总预算的20-30%
- 核心渠道: 总预算的50-60%
- 新渠道探索: 总预算的10-20%

## KOL/网红营销

### KOL分级

| 级别 | 粉丝量 | 特点 | 适用场景 |
|------|--------|------|----------|
| 头部KOL | >100万 | 影响力大，费用高 | 品牌曝光、大型活动 |
| 腰部KOL | 10-100万 | 性价比高，垂直精准 | 产品种草、口碑建设 |
| 尾部KOL | 1-10万 | 成本低，互动率高 | 铺量种草、UGC积累 |
| KOC | <1万 | 真实感强，信任度高 | 口碑传播、社区运营 |

### 合作流程
1. 市场调研: 确定目标市场和受众画像
2. KOL筛选: 匹配度、互动率、粉丝质量、历史合作效果
3. 方案沟通: Brief撰写、创意方向、交付标准
4. 内容审核: 品牌调性、合规性、信息准确性
5. 发布追踪: 数据监控、互动管理、效果评估
6. 复盘优化: ROI分析、长期合作评估

### 效果评估指标
- CPE (Cost Per Engagement) = 合作费用 / 互动总数
- CPV (Cost Per View) = 合作费用 / 观看总数
- 种草转化率 = 通过KOL内容产生的转化 / KOL内容曝光
- 品牌提及增长率

## 市场本地化

### 本地化清单
- 语言翻译: 专业翻译 + 本地母语审校 (非机翻)
- 文化适配: 颜色、符号、节日、禁忌
- 支付方式: 本地主流支付 (如东南亚的GrabPay/GCash)
- 法律合规: 数据隐私(GDPR/CCPA)、广告法规、年龄分级
- 客服本地化: 本地语言客服、时区覆盖

### 重点市场特征

#### 东南亚 (SEA)
- 主要市场: 印尼、泰国、越南、菲律宾、马来西亚
- 社交偏好: TikTok、Facebook、LINE(泰国)、Zalo(越南)
- 付费特点: 价格敏感，小额高频付费
- 文化要点: 宗教节日(斋月等)、本地明星效应

#### 日韩
- 社交偏好: Twitter/X(日本)、Naver/KakaoTalk(韩国)
- 付费特点: 付费意愿高，品质要求高
- 文化要点: 二次元文化(日本)、偶像文化(韩国)

#### 欧美
- 社交偏好: Instagram、TikTok、YouTube、Reddit
- 付费特点: 订阅制接受度高，注重隐私
- 文化要点: 多元化、包容性、环保意识

#### 中东
- 主要市场: 沙特、阿联酋、埃及
- 社交偏好: Snapchat(沙特)、Instagram、TikTok
- 付费特点: 高ARPU，土豪用户多
- 文化要点: 伊斯兰文化、阿拉伯语RTL排版、斋月营销

## 内容创作模板

### 小红书种草笔记模板

```
标题: {emoji}{产品/游戏名}+{核心卖点}+{情绪词}
示例: 🔥原神新版本太炸了！必看攻略帮你省1000💎

正文结构:
1. 开头hook (1-2句，制造共鸣或好奇)
2. 核心内容 (3-5个要点，配emoji)
3. 个人体验/评价 (真实感)
4. 行动号召 (引导互动)

标签: #主标签 #细分标签 #长尾标签 #热门话题 #品牌标签
```

### TikTok脚本模板

```
0-3秒: Hook (问题/冲突/视觉冲击)
3-15秒: 核心内容展示
15-25秒: 亮点/反转/高潮
25-30秒: CTA (关注/评论/链接)

配乐: 使用热门BGM
字幕: 必须添加，提升完播率
```

### 广告文案框架 (AIDA)

```
Attention (注意): 抓眼球的标题/画面
Interest (兴趣): 展示产品价值/痛点解决
Desire (欲望): 社交证明/限时优惠/独特卖点
Action (行动): 明确CTA (下载/购买/注册)
```

## 报告模板

出海营销报告应包含：
1. 市场概览: 目标市场数据、竞品动态
2. 渠道表现: 各平台核心指标及环比变化
3. 广告效果: 花费、CPI/CPA、ROAS、素材表现
4. 内容运营: 发布数据、互动率、爆款内容分析
5. KOL合作: 合作效果、ROI、后续计划
6. 优化建议: 渠道调整、预算重分配、素材迭代方向
7. 下期计划: 重点市场、营销节点、预算规划
