# 游戏玩家留存分析

为游戏运营团队提供玩家留存数据分析、流失预测和留存优化的完整工作流。

## 核心指标体系

### 留存率指标
- 次日留存率 (Day 1 Retention): 新用户次日回访比例，健康值 > 40%
- 7日留存率 (Day 7 Retention): 健康值 > 20%
- 30日留存率 (Day 30 Retention): 健康值 > 10%
- 滚动留存率: 第N天内任意一天回访即算留存

### 计算公式

```
留存率 = 第N天回访用户数 / 第0天新增用户数 × 100%
加权留存率 = Σ(各渠道留存率 × 各渠道用户占比)
```

### 生命周期价值 (LTV)

```
LTV = ARPU × 平均生命周期天数
LTV = ARPPU × 付费率 × 平均生命周期天数
简化LTV = 累计收入 / 累计新增用户数
```

### 关联指标
- ARPU (每用户平均收入) = 总收入 / 活跃用户数
- ARPPU (每付费用户平均收入) = 总收入 / 付费用户数
- 付费率 = 付费用户数 / 活跃用户数
- DAU/MAU 比值: 衡量用户粘性，健康值 > 20%

## 分析工作流

### 1. 留存趋势分析

输出留存率随时间变化的趋势，识别异常波动：
- 按日/周/月粒度绘制留存曲线
- 对比不同版本、不同渠道的留存差异
- 标注版本更新、活动上线等关键事件节点

### 2. 用户分群留存对比

按以下维度分群对比留存：
- 获客渠道 (自然量 vs 买量 vs 社交分享)
- 设备类型 (iOS vs Android vs PC)
- 地区 (国内 vs 海外各区域)
- 付费状态 (免费用户 vs 首充用户 vs 大R)
- 新手引导完成度 (完成 vs 未完成)

### 3. 流失预警模型

关键流失信号：
- 登录频率下降 > 50%
- 日均在线时长下降 > 40%
- 社交互动减少 (公会/好友)
- 连续3天未登录
- 体力/资源溢出未消耗

流失风险等级：
- 高风险: 7天未登录 + 历史活跃度高
- 中风险: 3-7天未登录 或 活跃度持续下降
- 低风险: 登录但参与度下降

### 4. 留存优化策略

#### 新手期 (Day 0-3)
- 优化新手引导流程，降低学习曲线
- 首日赠送强力角色/装备，提升爽感
- 设置明确的短期目标和奖励

#### 成长期 (Day 4-14)
- 开放社交系统 (公会、好友、组队)
- 引入每日任务和签到奖励
- 推送个性化内容推荐

#### 成熟期 (Day 15-30)
- 定期更新内容 (新关卡、新角色、新玩法)
- 举办限时活动和赛季
- 建立付费习惯 (月卡、通行证)

#### 回流策略
- 回流礼包 (7天/14天/30天未登录分级)
- Push通知 + 短信召回
- 版本大更新时集中召回

## 报告模板

分析报告应包含：
1. 数据概览: 核心留存指标及环比变化
2. 趋势分析: 留存曲线及异常标注
3. 分群对比: 不同维度的留存差异
4. 流失分析: 流失用户画像及原因
5. 优化建议: 具体可执行的改进方案
6. ROI预估: 预期留存提升及收入影响
