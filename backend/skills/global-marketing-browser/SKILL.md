# 出海营销浏览器数据采集

为出海营销团队提供基于AgentCore Browser的网页数据采集、竞品监控和市场调研能力。

## 核心能力

### 1. 竞品网站分析
- 访问竞品官网，提取产品信息、定价策略、促销活动
- 分析竞品网站结构、SEO策略、内容布局
- 监控竞品新品发布和功能更新

### 2. 社交媒体数据采集
- 浏览TikTok/Facebook/Instagram/LinkedIn/YouTube公开页面
- 采集热门帖子、互动数据、粉丝增长趋势
- 分析竞品社交媒体运营策略和内容风格
- 监控行业KOL动态和合作信息

### 3. 广告素材研究
- 访问Facebook Ad Library查看竞品广告素材
- 分析广告文案、视觉风格、投放策略
- 采集行业热门广告创意和趋势

### 4. 市场调研数据采集
- 访问行业报告网站获取市场数据
- 采集Google Trends搜索趋势
- 获取目标市场电商平台(Amazon/Shopee/Lazada)的品类数据
- 分析行业论坛和社区的用户反馈

## 浏览器使用工作流

### 竞品分析流程
1. 导航到竞品网站首页
2. 截取页面快照，分析整体布局和信息架构
3. 浏览产品页面，提取产品列表和定价
4. 检查社交媒体链接和客户评价
5. 生成竞品分析报告

### 社交媒体监控流程
1. 访问目标社交平台公开页面
2. 采集最新帖子内容和互动数据
3. 分析内容主题、发布频率、互动模式
4. 识别高互动内容的共同特征
5. 输出社交媒体运营洞察

### 广告研究流程
1. 访问Facebook Ad Library (facebook.com/ads/library)
2. 搜索竞品或行业关键词
3. 浏览活跃广告素材
4. 分析广告文案、CTA、视觉风格
5. 总结广告投放趋势和优化建议

## 数据采集注意事项
- 仅采集公开可访问的数据
- 遵守网站robots.txt和使用条款
- 采集频率合理，避免对目标网站造成负担
- 数据仅用于市场分析和策略参考
- 注意不同地区的数据隐私法规(GDPR/CCPA等)

## 输出格式

### 竞品分析报告
```
## 竞品分析: {竞品名称}
- 网站: {URL}
- 主要产品/服务: {列表}
- 定价策略: {描述}
- 社交媒体布局: {各平台链接和粉丝数}
- 内容策略: {分析}
- 优势/劣势: {对比分析}
```

### 社交媒体监控报告
```
## 社交媒体监控: {平台名称}
- 监控时间: {日期范围}
- 热门内容TOP5: {列表}
- 互动趋势: {分析}
- 内容主题分布: {分类}
- 运营建议: {基于数据的建议}
```
